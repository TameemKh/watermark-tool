📁 محتوى المجلد:
─────────────────
  app.py               ← الواجهة الرئيسية
  watermark_engine.py  ← محرك المعالجة
  requirements.txt     ← المكتبات المطلوبة
  run.bat              ← زر التشغيل
  logo.png             ← ملف اللوجو (ضعه هنا!)

──────────────────────────────────────────
🔧 التثبيت (مرة واحدة فقط)
──────────────────────────────────────────
1. تأكد أن Python مثبّت على جهازك
   يمكنك تحميله من: https://www.python.org/downloads/
   ✅ أثناء التثبيت فعّل خيار "Add Python to PATH"

2. ضع ملف اللوجو (PNG) بجانب app.py واسمه:
      logo.png

──────────────────────────────────────────
🚀 طريقة التشغيل
──────────────────────────────────────────
   انقر مرتين على:  run.bat
   سيفتح المتصفح تلقائياً بالأداة

──────────────────────────────────────────
📖 طريقة الاستخدام
──────────────────────────────────────────
1. ارفع ملفات PDF (يمكن رفع أكثر من ملف)
2. اختر نوع العلامة:
      🖼️  صورة      → اللوجو فقط
      ✏️  نص        → نص IVR TEAM
      🔀  نص+صورة  → كلاهما بالتناوب

3. اختر وضعية العلامة:
      📐  زاوية        → أسفل يمين كل صفحة
      ↗️  قطري         → متكرر بشكل مائل
      🔲  كامل الصفحة  → علامة كبيرة وسط الصفحة

4. اضبط الشفافية بالـ Slider
   (10% هو الافتراضي — خفي لكن مرئي)

5. اضغط "تطبيق العلامة المائية"
6. حمّل الملفات المعالجة

──────────────────────────────────────────
⚠️  ملاحظات
──────────────────────────────────────────
• الملفات المرفوعة لا تُحفظ على الجهاز
• إذا رفعت ملفاً واحداً → تحميل مباشر
• إذا رفعت أكثر من ملف → تحميل كـ ZIP
• لاستثناء صفحات معينة اكتب أرقامها
  مفصولة بفاصلة (مثال: 1, 2, 5)
